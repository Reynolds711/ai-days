Sector recipes

Decision prompt

Which agent pilot has the clearest outcome, user, grounding source, and review point?

Handoff rule

A safe agent knows when to stop. Name the requests it can prepare, the ones it can clarify, and the ones it must route to a person.

Constraint pattern

The agent can summarize, classify, draft, and recommend. It cannot approve, determine eligibility, make clinical judgments, or send without review.

Test pattern

Test a normal request, an ambiguous request, an overreach request, and an urgent request.
